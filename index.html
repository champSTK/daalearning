// 02 — Core Concepts
DAA Topics
The algorithmic paradigms powering every visualizer in this lab.
🔢
Sorting Algorithms
From brute-force O(n²) approaches like Bubble and Selection Sort to efficient O(n log n) divide-and-conquer algorithms.
O(n²) · O(n log n)
🔍
Pathfinding
BFS guarantees unweighted shortest paths; Dijkstra handles weighted graphs; A* uses heuristics to search efficiently.
BFS · Dijkstra · A*
🌐
Graph Algorithms
DFS, BFS traversals and shortest-path algorithms form the backbone of networking, AI, and systems design.
DFS · BFS · MST
↩
Backtracking
Systematic trial-and-error via state-space tree exploration. Prunes invalid branches early — powers N-Queens and Maze.
State Space Tree
📊
Dynamic Programming
Overlapping subproblems solved once and memoized. Powers 0/1 Knapsack, LCS, and many optimisation problems.
Memoization · DP Table
💡
Greedy Algorithms
Make locally optimal choices at each step. Activity Selection, Huffman Coding, and Fractional Knapsack are classics.
Local Optimal Choice
🔤
String Algorithms
Efficient pattern matching with KMP O(n+m) vs naive O(nm). The failure function enables intelligent skip-ahead.
KMP · Failure Function
🧩
Divide & Conquer
Break problems into subproblems, solve recursively, combine results. Powers Merge Sort, Quick Sort, Binary Search.
Recursion · Combine
// 03 — Learning Flow
How to Use This Lab
A structured path from theory to hands-on mastery.
01
Understand the Concept
Read the algorithm description and complexity analysis in the Concepts section. Understand why an algorithm exists and its theoretical efficiency bounds before opening a visualizer.
02
Visualize the Algorithm
Open a project and watch the algorithm execute step-by-step. Use speed control to slow it down. Observe data structure changes, comparisons, and backtrack decisions live.
03
Interact & Experiment
Tweak parameters — change array sizes, add obstacles, build custom graphs. Compare multiple algorithms side by side and form hypotheses you can test live.
